# generator-vintage-frontend

[![NPM version][npm-image]][npm-url]
![][license-url]

Yeoman generator for modern front-end apps.

## Prerequisites ##

##### Install [yeoman](http://yeoman.io/):
`npm install -g yo`

##### Install [generator-vintage-frontend](https://www.npmjs.com/package/generator-vintage-frontend):
`npm install -g generator-vintage-frontend`

> prepend with `sudo` if you are running on `*nix` OS.

## Use generator-vintage-frontend ##

`yo vintage-frontend`

> Answer questions

![](https://raw.githubusercontent.com/Vintage-web-production/generator-vintage-frontend/master/install.png)

Done!

## After installation ##

Start development

```
npm run development
```

Build production bundle

```
npm run production
```
